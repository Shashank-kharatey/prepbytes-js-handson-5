Q.1 You are given a string S, and your task is to return an array B(having a size of 2), where B[0] contains the count of character A (uppercase) in string S and B[1] contains the count of character D (uppercase) in string S. Note: You have to complete countCharacters function. No need to take any input.

Ans.1 const countCharacters = (S) => 
        {
          var B=[];
          var counta=0;
          var countd=0;
           for (i=0; i < S.length;i++)
           {
             if (S.charAt(i)==="A")
             {
               counta++;
             }
             if (S.charAt(i)==="D")
             {
               countd++;
             }
           }
           B[0]=counta;
           B[1]=countd;
           return B;
        };

Q.4 You are given an array A containing N integer elements, and your task is to return an array B(having a size equal to 2), where B[0] contains the sum of all even elements of array A and B[1] has the sum of all odd elements of the array A. Note: You have to complete findEvenOdd function. No need to take any input.

Ans.4 const findEvenOdd = (N, Arr) => 
        {
          var c=[0,0];
            for (i=0; i < N; i++)
            {
              if (Arr[i]%2===0)
              {
                c[0]+=Arr[i];
              }
              else
              {
                c[1]+=Arr[i];
              }
            }
            return c;
        };

Note: You have to complete isAllPass function. No need to take any input.

Now, he wants to know how many M unique color shirts are left in the wardrobe. Prepbuddy wants you to find M. Note: As there are many shades of a color so consider integers as a color name. i.e, color of shirt is 0,1,2, … , N.

Ans.9 function Unique_Shirts (arr,N) {
         var c=0;
        for (i=0; i < N; i++)
        { var a=0;
          for (j=0; j < N; j++)
          {
            if (arr[i]==arr[j])
            {
              a++;

Q.10 Congratulations on making up to this question. Let us give you a fairly simple array problem to solve. If you know how to iterate through the array, you will easily be able to solve this. The problem statement is simple- given N elements, find the minimum and maximum numbers among those elements.

Ans.10 function arrayMin(arr) {
        var a=arr[0];
          for (j=1; j < arr.length; j++)
          {
            if (a>arr[j])
            {
              a=arr[j];
            }
          }
        return a;
      }
      
      function arrayMax(arr) {
        var b=arr[0];
          for (j=1; j < arr.length; j++)
          {
            if (b < arr[j])
            {
              b=arr[j];
            }
          }
        return b;
      }
